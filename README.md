### 安装mysql
1、修改my-default.ini
2、添加
	basedir=D:\Program Files\MySql\mysql-5.7.17-winx64
	datadir=D:\Program Files\MySql\mysql-5.7.17-winx64\data
3、mysqld -install
4、mysqld --initialize 
5、net start mysql
6、查询data目录下的*.err最后会有一个默认的密码就可以了
